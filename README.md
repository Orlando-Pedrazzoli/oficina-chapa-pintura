# Street Paint - Site da Oficina

Site desenvolvido em React para a oficina Street Paint de Sintra.

## 🚀 Como Rodar o Projeto

### Instalação

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

O site abrirá em `http://localhost:5173`

## 📁 Estrutura

```
src/
├── components/     # Componentes React
├── pages/         # Páginas (Home, Services, Contact)
├── assets/        # Imagens e recursos
├── App.jsx        # Componente principal
└── main.jsx       # Entry point
```

## 🛠️ Tecnologias

- CSS3
